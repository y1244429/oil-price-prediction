## 💡 使用说明

### 1. 查看真实数据
网站会自动使用AKShare获取的Brent原油价格和PMI数据。

### 2. 数据来源标识
API响应中包含`data_source`字段:
```json
{
  "status": "success",
  "data_source": "AKShare",  // 或 "Mock"
  "data": {...}
}
```

### 3. 手动刷新数据
访问 `/api/data` 会自动获取最新数据。
