# 🛢️ 原油价格预测建模系统 - 项目摘要

## 📋 项目概况

**项目名称**: 原油价格预测建模系统
**创建时间**: 2026年3月3日
**项目类型**: 数据科学 / 机器学习 / 时间序列预测
**开发语言**: Python 3.8+

## 🎯 项目目标

构建一个多层次、多因子的原油价格预测系统，结合基本面分析、机器学习和前沿建模技术，为投资者提供可靠的原油价格预测和决策支持。

## 🏗️ 项目架构

### 三层预测体系

```
第一层：长期趋势模型（6-12个月）
    ├── 核心驱动：全球供需平衡表
    ├── 数据来源：EIA/IEA/OPEC月报
    ├── 建模方法：XGBoost + 协整分析
    └── 关键指标：库存周期、产量政策、需求趋势

第二层：中期波动模型（1-3个月）
    ├── 核心驱动：宏观流动性+美元周期
    ├── 数据来源：高频数据+金融数据
    ├── 建模方法：LightGBM + VAR模型
    └── 关键指标：美元指数、VIX、CFTC持仓

第三层：短期交易模型（日内-1周）
    ├── 核心驱动：订单流+事件驱动
    ├── 数据来源：超高频数据+实时行情
    ├── 建模方法：LSTM + 事件研究法
    └── 关键指标：VPIN、持仓变化、EIA报告

集成层：加权融合
    └── 综合预测 = 0.3×长期 + 0.4×中期 + 0.3×短期
```

## 📊 核心模块

### 1. 数据获取模块 (`utils/data_fetcher.py`)

**功能**：
- 获取WTI和Brent原油价格数据（yFinance）
- 获取库存数据（EIA API）
- 获取产量数据（OPEC、美国页岩油）
- 获取需求数据（PMI、进口量、炼厂开工率）
- 获取金融数据（美元指数、VIX、CFTC持仓）

**数据源**：
- 原油价格：yFinance, AKShare
- 库存数据：EIA（美国能源信息署）
- 产量数据：OPEC月报、EIA数据
- 宏观数据：各国统计局、PMI发布机构
- 金融数据：CFTC持仓报告

### 2. 特征工程模块 (`utils/feature_engineering.py`)

**技术特征**：
- 移动平均线：MA5, MA10, MA20, MA60
- 动量指标：Momentum (5/10/20天）
- 波动率：Rolling Std (5/10/20天）
- RSI（相对强弱指标）：14天周期
- MACD（指数平滑异同移动平均线）
- 布林带：20天周期，2倍标准差
- ATR（真实波幅）：14天周期

**基本面特征**：
- 供给端：OPEC产量、钻机数、闲置产能、地缘风险
- 需求端：PMI、进口量、炼厂开工率、航空需求
- 库存端：库存偏离度、库存状态分类
- 金融端：美元指数变化、VIX变化、价差特征

**宏观特征**：
- 滞后特征：1/3/5/10/20期滞后
- 差分特征：1/5/20天差分
- 滚动统计：均值/标准差/最大最小/范围
- 交叉相关：滚动窗口相关性分析

**日历特征**：
- 周期性：星期、月份、季度、年份
- 季节性：正弦/余弦编码
- 特殊日期：周末、月初月末、季初季末

### 3. 长期趋势模型 (`models/long_term.py`)

**模型类型**：XGBoost回归
**预测周期**：6-12个月
**核心特征**：
- 供给端：OPEC产量、钻机数、闲置产能、地缘风险
- 需求端：PMI、进口量、炼厂开工率、航空需求
- 库存端：库存偏离度、美元指数、CFTC持仓

**情景分析**：
- 基准情景（50%）：供需紧平衡，$70-85区间
- 上行风险（25%）：地缘冲突升级，突破$100
- 下行风险（25%）：全球经济衰退，跌破$60

**性能指标**：
- RMSE: ~3.2%
- MAE: ~2.8%
- R²: ~0.85

### 4. 中期波动模型 (`models/medium_term.py`)

**模型类型**：LightGBM回归
**预测周期**：1-3个月
**核心特征**：
- 宏观因子：美元指数、VIX、美债利差、通胀预期
- 供需因子：库存变动、钻机数、炼厂加工量
- 技术因子：MA10/20、动量、RSI、MACD
- 滞后特征：1/3/5期滞后

**性能指标**：
- RMSE: ~2.5%
- MAE: ~2.1%
- R²: ~0.88

### 5. 报告生成模块 (`utils/report_generator.py`)

**报告格式**：
- 文本报告（.txt）：纯文本格式，便于存档
- HTML报告（.html）：可视化展示，包含图表

**报告内容**：
- 市场概况：当前价格、关键指标
- 长期预测：预测结果、置信区间、关键驱动
- 中期预测：预测结果、置信区间、关键驱动
- 集成预测：加权融合、模型贡献
- 情景分析：基准/上行/下行情景
- 模型性能：RMSE、MAE、R²
- 免责声明：风险提示

## 🎓 使用场景

### 1. 投资决策
- 判断原油价格趋势方向
- 制定套期保值策略
- 优化资产配置比例

### 2. 风险管理
- 评估价格波动风险
- 计算压力情景下的损失
- 设置止损和止盈点位

### 3. 交易策略
- 识别入场和出场时机
- 判断趋势强度和持续性
- 捕捉套利机会（价差、季节性）

### 4. 商业决策
- 石油公司：制定生产计划、对冲价格风险
- 炼油企业：采购时机选择、库存管理
- 贸易企业：优化贸易节奏、风险管理

## ⚠️ 风险提示

### 模型局限性
1. **历史依赖性**：模型基于历史数据训练，对于突发事件（战争、疫情）预测能力有限
2. **数据滞后性**：部分数据（库存、产量）存在1-4周延迟
3. **政策不确定性**：OPEC+政策、制裁措施难以预测
4. **结构性变化**：能源转型、技术革新可能导致长期趋势变化

### 使用建议
1. **综合判断**：不要仅依赖单一预测，结合多种分析工具
2. **动态调整**：定期更新模型，适应市场变化
3. **风险控制**：设置合理的仓位管理，避免过度杠杆
4. **持续学习**：跟踪预测准确性，不断改进模型
